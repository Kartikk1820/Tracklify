# 💰 Expense Tracker

A **Personal Finance Tracker** web application built with **React.js, Express.js, and MongoDB**.  
It helps users manage their expenses and income by tracking transactions with categories, dates, and totals.

---

## ✨ Features

- 📊 **Dashboard Overview**
  - Total Balance, Total Income, and Total Expense at a glance.
- 📝 **CRUD Operations**
  - Add new transactions (income/expense).
  - Edit existing transactions with pre-filled details.
  - Delete transactions with confirmation.
- 🔍 **Transaction List**
  - Displays title, category, amount, date, and actions.
  - Color-coded amounts: 🟢 income, 🔴 expense.
- 🔒 **Validation**
  - Prevents empty fields and ensures valid data.
- 🎨 **UI/UX**
  - Gradient black background for a professional look.
  - Responsive and user-friendly design.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Styling:** CSS (custom components + gradient theme)
